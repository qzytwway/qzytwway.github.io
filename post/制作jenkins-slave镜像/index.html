制作jenkins Slave镜像

1 min read

构建镜像需要的包以及软件如下图：

已经打成tar包供下载，下载链接如下。

链接：https://pan.baidu.com/s/1T2uBx3fm7FlnFmiFd5aGIA 
提取码：6666

Dockerfile

FROM jenkins/inbound-agent

USER root

WORKDIR /home/jenkins

ADD  apache-maven-3.8.5-bin.tar.gz /usr/local/
COPY helm kubectl /usr/bin/
COPY settings.xml /usr/local/apache-maven-3.8.5/conf/settings.xml

ENV MVN_HOME=/usr/local/apache-maven-3.8.5
ENV PATH=${PATH}:${JAVA_HOME}/bin:${MVN_HOME}/bin

ENTRYPOINT ["jenkins-slave"]

使用docerkbuile命令构建镜像，之后可以使用docker tag更改镜像名上传到阿里云

docker build -t my-jenkins-slave:v1 .
